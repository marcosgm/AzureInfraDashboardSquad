# Squad Team

## Project Context

**Project:** Azure Infrastructure Visualizer
**Description:** A real-time app that displays Azure infrastructure with interactive logical diagrams. Dashboard-like charts show uptime, latency, and usage metrics. Connects to Azure read-only via the Azure MCP server for discovery and specific APIs for telemetry. Maintains a metric cache refreshed every 5 minutes.
